# To the power of

Create a function that takes a base and an exponent and returns the result.

## Examples

СalculateExponent(5, 5) ➞ 3125

СalculateExponent(10, 10) ➞ 10000000000

СalculateExponent(3, 3) ➞ 27

## Goals

- [ ] Take a base and an exponent as arguments.
- [ ] Return the result.
- [ ] Handle bad inputs.
